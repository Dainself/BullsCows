A well-known logic game in numerical version. You can find game rules there https://en.wikipedia.org/wiki/Bulls_and_Cows

Press Start to run the game. You can launch new game at any moment. At the top of window are displayed turns and current status with tips. On the left side CPU try to guess your number. Enter num of bulls/cows according to game rules. On the right side you try to guess CPU's number. You can make best conclusions by viewing the game history, which displayed at textfield below. Clicking loupe icon provides a more easily view.

You can firstly enter answer at left side, or at right side, no matter. But you cannot twice in a row enter an answer to the same side.
